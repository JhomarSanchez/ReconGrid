Este modo está diseñado para procesar archivos que contienen una columna con información de vehículos en formatos específicos.

Formato esperado:

RENAULT-12(1970,1990); RENAULT-18 GTL (1978-1981)
💡 Nota: Esta columna generalmente se denomina VEHICULO MASTER en los archivos de origen.

¿Qué hace el sistema?

  • Procesa la información de la columna seleccionada
  • Extrae marca, modelo y años de los vehículos
  • Retorna un archivo Excel nuevo con las columnas organizadas correctamente (solo si los datos son válidos)

Este modo se encarga de comparar la información entre dos documentos de Excel.

¿Cómo funciona?

  1. Selecciona los dos archivos Excel que deseas comparar
  2. Elige las columnas en las que deseas buscar información coincidente
  3. Si la información coincide entre ambos documentos, se extrae la información correspondiente que hayas elegido

Los códigos pueden ser extraidos si tienen ligeras variaciones, por ejemplo si existe un código ABC-123. Se buscará si existe un código:

ABC 123, ABC123, ABC/123, ABC//123, ABC\123
💡 Ejemplo: Si tienes un archivo con códigos de producto y otro con descripciones, puedes comparar por código y extraer automáticamente las descripciones correspondientes.

Como su nombre indica, este modo compara dos columnas dentro de la misma hoja de Excel.

Resultados:

  • Si las columnas coinciden exactamente → Se indica como Coincide
  • Si las columnas no coinciden → Se indica como Difiere
💡 Resultado: La información de coincidencia se mostrará en una nueva columna agregada al archivo.
