ReconGrid se concentra en conciliacion tabular y comparacion sobre Excel.

Comparar archivos

Cruza dos libros de Excel, normaliza identificadores y genera un archivo de resultados auditable.

||

Comparar columnas internas

Evalua dos columnas dentro de una misma hoja y marca donde los valores coinciden o difieren.

Guia de modos de trabajo

Usa esta referencia rapida para decidir que flujo debes abrir.

Compara informacion entre dos documentos de Excel y extrae los datos asociados cuando encuentra coincidencias.

Pasos generales:

  1. Selecciona ambos archivos
  2. Configura hojas, encabezados y columnas clave
  3. Genera un archivo de salida con el resultado del cruce
ABC-123 | ABC 123 | ABC/123 | ABC123
ReconGrid normaliza identificadores para tolerar cambios de separadores, espacios y formatos frecuentes.

Contrasta dos columnas dentro de una misma hoja para validar consistencia antes de exportar o continuar con el analisis.

Resultados:

  • Marca filas con coincidencia exacta
  • Marca filas con diferencia
  • Agrega una nueva columna con el veredicto
Este flujo es util para conciliaciones internas, verificaciones previas y controles operativos.
